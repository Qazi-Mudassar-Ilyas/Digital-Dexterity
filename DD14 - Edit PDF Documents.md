# Edit PDF Documents

## The Idea
Many people think PDFs cannot be edited without expensive software. Microsoft Word can open and edit many PDF files, making it easy to update text or reuse existing content.

## Why It Matters
Instead of recreating a document from scratch or paying for specialized PDF editing software, you can quickly make changes using a tool you already have.

## Try It
1.	Open Microsoft Word.
2.	Click File → Open and select the PDF document.
3.	Word will convert the PDF into an editable document.
4.	Make the required changes.
5.	Save it as a Word document or export it again as a PDF using File → Save As.

## Pro Tip
This method works best with text-based PDFs. If the PDF is a scanned document, it may require Optical Character Recognition (OCR) before the text can be edited.

## Works On
Microsoft Word (Windows, Mac, and Microsoft 365)
